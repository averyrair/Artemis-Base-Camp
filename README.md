# Artemis Base Camp
### A cozy sandbox city-builder
The concept is that you're starting the first permanent settlement on the moon. You start in a tiny lunar lander and you have to go collect moon rock and ice and stuff to build a base.

It'll be kinda a sandbox game where you just build a cool looking base. But the grand vision is that eventually more astronauts will come to live at the base and you'll have to make sure everyone has what they need. So in that way it will kind of be a cross between a city builder and a Stardewlike. Planning to have several systems for keeping people happy and healthy like farming/nutrition/etc.
